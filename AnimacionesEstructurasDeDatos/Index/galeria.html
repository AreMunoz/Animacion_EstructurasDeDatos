Stack

Pila

Es de tipo LIFO

Para el manejo de los datos se cuenta con dos operaciones básicas: apilar (push), que coloca un objeto en la pila, y su operación inversa, retirar (o desapilar, pop), que retira el último elemento apilado.

Queue

Cola

Estructura FIFO (del inglés First In First Out)

Es una secuencia de elementos en la que la operación de inserción PUSH se realiza por un extremo y la operación de extracción POP por el otro.

List

Lista Enlazada

Se representa internamente como Nodos.

Agregar un elemento es fácil, o poco costoso en términos de alocación de memoria. Símplemente debemos alocar espacio para un nuevo nodo, y actualizar el ultimo nodo, para que apunte al nuevo, como "siguiente".
